## Requirements
* Python 3.7 <br>
* Pytorch 1.8.1 <br>
* Torchvision 0.9.1 <br>
* Cuda 11.0

# Usage
This is the Pytorch implementation of CBNet. It has been trained and tested on Linux(Win10 + Cuda 11 + Python 3.7 + Pytorch 1.8.1). 

## Training
* Copy the images under data/dataset/
* Run `main.py`
